# ChatbotAssignment

This is a simple mini project on creating a chatbot using dialogflow api. Here chatbot is about weather. It uses weather api also. 
This is a web application where query writen by user is sent to dialogflow api which replied based on that query.
After making the application, we deploy it on heroku plateform and get a link that is given in main repository.
